# Movies Reservation API 

## Description
#### For cinema owners, you can:
Create user, authenticate, create movies with posters, create sessions, manage seats, create and manage cinema room layouts.

#### For normal users, you can:
Create account, authenticate, buy seats in specific movies sessions, manage available seats,

---

## Requirements
### - Docker

---

## How to install
Clone the code to local and run `docker compose up`.
